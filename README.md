# 嵌入式应用软件模块化解耦与自动注册框架 (App-Core)

本项目是一个轻量级、高度解耦的嵌入式 Linux 应用框架。它深度借鉴了 **Linux 内核驱动模型 (`__initcall` 机制)**，专门为资源受限的嵌入式 Linux 应用程序设计，解决了传统开发中模块间高度耦合、构建耗时、宏污染等核心痛点。

---

## 核心架构原则

1. **机制与策略分离 (Mechanism vs Policy)**
   - 核心层 (`core/`) 只提供初始化时序控制与通用对象寄存代理的“机制”，不感知具体的业务实现（如 WiFi、Camera 等）。
2. **编译依赖倒置 (Compilation Dependency Inversion)**
   - 核心层在编译期 100% 不依赖应用层或业务模块的任何头文件。修改应用层逻辑绝对不会导致核心层代码重新编译。
3. **动态自愈与安全降维**
   - 通过 CMake 配置进行文件级裁剪。若某模块被关闭编译，主程序获取其接口指针时天然得到 `NULL`。主程序仅需通过简单的防空指针检查即可平滑降维，不会引发链接或运行期崩溃。

## 项目目录结构

```text
├── core/                  # 核心框架层（闭源/不常修改，无业务污染）
│   ├── sys_core.h         # 机制总线契约（采用通用整型引脚阻断依赖）
│   └── sys_core.c         # 机制总线实现（使用内部固定槽位安全数组）
│
├── app/                   # 应用业务层（日常频繁开发区域）
│   ├── app_modules.h      # 整个团队的【模块引脚总账本】
│   ├── main.c             # 主程序入口，面向接口与防空指针编程
│   └── features/          # 各解耦的功能特性模块
│       ├── CMakeLists.txt # 特性编译管理文件，接收 CMake 裁剪开关
│       ├── wifi_mod.h / .c
│       └── camera_mod.h / .c
│
├── CMakeLists.txt         # 顶层 CMake 配置文件，提供全局配置项与优化标志
├── .gitignore             # Git 忽略规则
└── .clang-format          # C 代码格式化标准 (Google C)
```

## 编译与运行指南

本项目提供了便捷的 [build.sh](file:///home/loong/demo/module_init/build.sh) 一键编译脚本，并支持自动导出 IDE 符号跳转数据库。

### 1. 查看编译帮助说明
```bash
./build.sh --help
```

### 2. 默认全功能编译与运行
直接运行脚本即可编译（默认自动启用 WiFi 和 Camera，并于根目录生成 `compile_commands.json` 软链接）：
```bash
./build.sh
./build/app_core
```

### 3. 模块选择性裁剪编译
支持在命令行传参进行模块编译裁剪。例如，清空构建历史并禁用 WiFi 模块（仅编译 Camera 模块）：
```bash
./build.sh --clean --wifi off --camera on
```
运行后输出会自动切换为网络降维模式，不包含任何 WiFi 驱动的物理加载：
```text
--- 应用层业务逻辑开始运转 ---

[Main业务] 提示：当前 WiFi 模块未启用，跳过网络业务。
[Camera驱动] 开启视频流成功！参数: 30 FPS, 分辨率: 1920x1080
```

## 工业级优化注意点

1. **链接期优化 (LTO)**
   - 由于本架构使用了函数指针间接调用，在 Release 编译模式下，CMakeLists 默认会启用 `-flto` (Link-Time Optimization) 以消除 CPU 分支预测上的间接调用开销。
2. **多线程并发安全**
   - `do_initcalls()` 是**单线程同步**顺序拉起的。长耗时、阻塞的任务（如复杂的网络物理握手、长探测等）请在初始化函数中拉起独立的后台线程异步执行，不可在初始化流程中死循环阻塞。
3. **防空指针红线**
   - 业务层使用 `sys_subsystem_get()` 索要任何模块指针后，**必须进行 `if (ptr != NULL)` 检查**后再调用其接口，这是保障整机在模块裁剪时绝不崩溃的防火墙。
